隱私權政策

這份政策對應《念有所歸》App 內的隱私權說明。 我們只處理你主動建立或選擇的內容，用來完成念誦、迴向、進度、提醒、備份與同步等功能。

摘要

你的資料主要留在裝置上，只有你主動開啟的服務才會離開本機。

App 內的資料會用於帶出預設值、統計進度、安排提醒、建立備份， 以及在可用時透過 iCloud 讓同一 Apple ID 的裝置延續使用。

1. 我們蒐集哪些資料

只蒐集你在 App 裡主動建立、選擇或匯出的內容。

本機使用資料

  • 偏好設定，例如字體大小、誦速、背景音、音量與提醒設定。
  • 迴向對象、目標、念誦紀錄與流程草稿。
  • 你主動選取的頭像照片與自訂備註。

匯出與備份資料

  • 你自行建立的 JSON 備份檔。
  • 你匯出的修持報表 HTML 檔。
  • 你主動分享的截圖或檔案。

2. 資料如何使用

資料只為了讓 App 內功能更完整，不拿去做廣告或跨 App 追蹤。

功能運作

  • 帶出預設經文、背景音與常用迴向對象。
  • 統計今日、本月、本年進度與修持報表。
  • 安排通知提醒，讓目標可以依週期提醒你。

使用者可見的效果

  • 顯示願樹、完成結果與歷史紀錄。
  • 支援離線使用與續誦。
  • 在可用時提供跨裝置同步。

3. 是否與第三方分享

不會把你的資料賣給廣告商，也不會做第三方分析投放。

Apple 系統服務

  • iCloud 同步：當你啟用且裝置可用時，資料會在你的 private CloudKit 資料庫中延續。
  • App Store / StoreKit：用於買斷功能驗證與交易處理。
  • 通知與照片選取：由 Apple 系統層級服務處理。

離開裝置的情況

  • 你主動匯出備份檔時。
  • 你主動分享截圖、檔案或報表時。
  • 同一 Apple ID 裝置之間透過 iCloud 同步時。

4. 你的控制權

你可以隨時調整、刪除、匯出或關閉這些資料與服務。

你可以做的事

  • 編輯或刪除迴向對象、目標、紀錄與草稿。
  • 自行匯出備份，或還原先前的備份。
  • 在 App 內關閉通知、背景音或 iCloud 同步。

刪除 App 後的影響

  • 本機資料會移除。
  • 若曾啟用 iCloud，同步資料仍可能依 Apple 雲端服務機制暫時保留。
  • 之後重新安裝，仍可透過同步或備份恢復資料。
